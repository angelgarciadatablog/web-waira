## 📸 Cómo Subir Imágenes

Para cada producto, necesitas subir las imágenes con esta nomenclatura:

```
assets/img/productos/NOMBRE-PRODUCTO/
├── 1.webp  ← OBLIGATORIA (aparece en catálogo)
├── 2.webp  ← Opcional (galería)
├── 3.webp  ← Opcional (galería)
├── 4.webp  ← Opcional (galería)
└── ...hasta 8.webp
```

### **Ejemplo práctico: Subir imágenes de SIENA-NEGRO**

```bash
# Opción 1: Copiar desde tu computadora
cp ~/Descargas/siena-negro-frente.jpg assets/img/productos/SIENA-NEGRO/1.jpg
cp ~/Descargas/siena-negro-lateral.webp assets/img/productos/SIENA-NEGRO/2.webp
cp ~/Descargas/siena-negro-detalle.png assets/img/productos/SIENA-NEGRO/3.png

# Opción 2: Arrastrar archivos (en Finder/Explorador)
# 1. Abre la carpeta: assets/img/productos/SIENA-NEGRO/
# 2. Arrastra tus imágenes
# 3. Renómbralas a: 1.webp, 2.webp, 3.webp, etc.
```

## 💡 Tips Rápidos

### **Si tienes muchas imágenes que organizar:**

1. Crea subcarpetas temporales por producto
2. Selecciona las mejores fotos (máximo 8)
3. Renombra en orden de importancia
4. Mueve a la carpeta final

### **Formatos recomendados:**

- ✅ **WebP** (mejor compresión, calidad alta)
- ✅ **JPG** (fotos, buena compatibilidad)
- ✅ **PNG** (si tienes transparencias)

### **Optimización:**

Si las imágenes son muy pesadas (>500KB), considera optimizarlas:
- Online: [TinyPNG](https://tinypng.com/), [Squoosh](https://squoosh.app/)
- Local: Herramientas como ImageMagick o cwebp
